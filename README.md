# DC Food Tour

A curated guided food tour website for experiencing the best of Columbia Heights & U Street in Washington, DC.

## Features

- **Interactive Booking Form** - Request tours with custom preferences
- **Cuisine Preferences** - Tours customized based on your favorite food styles
- **Group Flexibility** - Groups of 4-8 people
- **Pricing** - $85 per person
- **What's Included** - 4 stops featuring 3 courses + dessert with beverages

## Tour Details

- **Duration:** 3-3.5 hours
- **Area:** Columbia Heights & U Street, NW DC
- **Distance:** ~1 mile walking
- **Best Times:** Friday-Saturday, 6-8 PM
- **Price:** $85/person (4-8 person minimum)

## Built With

- Next.js 14 (App Router)
- React
- TypeScript
- Tailwind CSS
- Vercel (hosting)

## Local Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Open http://localhost:3000
```

## Deployment

This site is deployed on Vercel and automatically updates when you push to GitHub.

## Customization

- Edit restaurant details in `/app/page.tsx`
- Update email in footer for bookings: `info@dcfoodtour.com`
- Modify colors and styling with Tailwind classes
